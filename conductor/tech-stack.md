# Tech Stack: ThoughtsManager

## Frontend Framework
- **Next.js (React):** For a robust, performant web application structure.
- **Tailwind CSS:** For efficient, utility-first styling.
- **React Flow:** For the interactive graph visualization and spatial canvas.

## Data Storage & State Management
- **Zustand:** Lightweight and fast state management for the application's UI and graph state.
- **Dexie.js (IndexedDB):** A clean wrapper for IndexedDB to handle local-first data persistence.

## AI & Semantic Discovery
- **Ollama API:** Direct integration with locally running Ollama instances for LLM capabilities.
- **Voy / Orama:** Local vector search libraries for fast, browser-based semantic discovery and embeddings management.

## Testing & Quality Assurance
- **Vitest:** A fast, Vite-native unit testing framework.
- **React Testing Library:** For testing component interactions and accessibility.

## Development Tools
- **TypeScript:** For type safety and improved developer experience.
- **ESLint & Prettier:** For code quality and consistent formatting.
